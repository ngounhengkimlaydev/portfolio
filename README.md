## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/ngounhengkimlaydev/portfolio
```

Navigate to the project:

```bash
cd portfolio
```

Install dependencies:

```bash
pnpm install
```

Run development server:

```bash
pnpm dev
```

Build for production:

```bash
pnpm build
```
